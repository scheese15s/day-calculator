# day-calculator

Family D-Day mini app for Apps in Toss and web.

## Local development

```bash
npm install
npm run dev
```

Node.js `20.x` 이상 또는 `22.x` 이상이 필요합니다.

## Apps in Toss

The app name is `day-calculator`.

After starting the dev server, open the sandbox app with:

```text
intoss://day-calculator
```
